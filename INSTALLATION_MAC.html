Dars Manager - Installation sur Mac

Les audios, analyses et exports restent localement sur votre Mac.

Installation normale

  1. Dezippez le fichier recu.
  2. Ouvrez le dossier dars-manager.
  3. Double-cliquez sur install_macos.command.
  4. Si macOS affiche une alerte simple, faites clic droit sur le fichier, puis Ouvrir.

Si macOS dit que le fichier est endommage

Ne cliquez pas sur Placer dans la corbeille. Cliquez sur Annuler.

Ensuite:

  1. Ouvrez l'application Terminal.
  2. Copiez-collez la commande ci-dessous.
  3. Appuyez sur Entree.
cd ~/Downloads/dars-manager-v0.9 && xattr -dr com.apple.quarantine . && chmod +x install_macos.command drsm_mac.command && ./install_macos.command

Si le dossier n'est pas dans Downloads

  1. Ouvrez Terminal.
  2. Tapez cd avec un espace apres cd.
  3. Glissez le dossier dars-manager dans Terminal.
  4. Appuyez sur Entree.
  5. Copiez-collez cette commande:
xattr -dr com.apple.quarantine . && chmod +x install_macos.command drsm_mac.command && ./install_macos.command

Apres installation

Un raccourci est cree sur le Bureau:

Dars Manager.command

Les fichiers sont stockes dans:

Documents/DarsManager
