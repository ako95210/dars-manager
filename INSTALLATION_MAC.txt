DARS MANAGER - INSTALLATION SUR MAC
===================================

Ce programme reste local sur votre Mac.
Vos audios, analyses et exports ne sont pas envoyes dans le cloud.


SI VOUS AVEZ UN FICHIER .PKG
----------------------------

1. Double-cliquez sur:

   dars-manager-vX.Y-mac.pkg

2. Suivez l'assistant d'installation.

3. Apres installation, ouvrez:

   Applications > Dars Manager

ou utilisez le raccourci cree sur le Bureau:

   Dars Manager.command


INSTALLATION
------------

Cette section concerne seulement le ZIP, si vous n'avez pas de fichier .pkg.

1. Dezippez le fichier recu.

2. Ouvrez le dossier dars-manager.

3. Vous pouvez d'abord ouvrir:

   INSTALLATION_MAC.html

   Ce fichier affiche les memes instructions dans le navigateur avec une commande facile a copier.

4. Double-cliquez sur:

   install_macos.command

5. Si le Mac refuse d'ouvrir le fichier avec le message:

   "install_macos.command est endommage et ne peut pas etre ouvert"

   Ne cliquez pas sur "Placer dans la corbeille".
   Cliquez sur "Annuler".

   Ensuite, suivez la section:

   SI LE MAC DIT QUE LE FICHIER EST ENDOMMAGE

6. Si le Mac affiche seulement une alerte de securite:

   - faites clic droit sur "install_macos.command";
   - cliquez sur "Ouvrir";
   - confirmez avec "Ouvrir".

7. Une fenetre Terminal peut s'ouvrir.
   Laissez l'installation se terminer.
   Le premier lancement peut prendre plusieurs minutes.

8. Quand l'installation est terminee, un raccourci est cree sur le Bureau:

   Dars Manager.command


SI LE MAC DIT QUE LE FICHIER EST ENDOMMAGE
------------------------------------------

1. Ouvrez l'application Terminal.

2. Copiez-collez cette commande:

   cd ~/Downloads/dars-manager-v0.9 && xattr -dr com.apple.quarantine . && chmod +x install_macos.command drsm_mac.command && ./install_macos.command

3. Appuyez sur Entree.

Si le dossier n'est pas dans Downloads ou ne s'appelle pas dars-manager-v0.9:

1. Tapez:

   cd 

   Attention: laissez un espace apres cd.

2. Glissez le dossier dars-manager dans la fenetre Terminal.

3. Appuyez sur Entree.

4. Tapez:

   xattr -dr com.apple.quarantine . && chmod +x install_macos.command drsm_mac.command && ./install_macos.command

5. Appuyez sur Entree.


UTILISATION
-----------

Pour lancer Dars Manager:

1. Double-cliquez sur le raccourci du Bureau:

   Dars Manager.command

2. L'application s'ouvre dans votre navigateur.

3. Si le navigateur ne s'ouvre pas automatiquement, ouvrez cette adresse:

   http://localhost:8501


OU SONT LES FICHIERS ?
----------------------

Vos fichiers sont stockes ici:

   Documents/DarsManager

Ce dossier contient:

   uploads    : audios charges
   analyses   : analyses sauvegardees
   exports    : audios generes
   hf_cache   : cache du modele Whisper


SI PYTHON EST DEMANDE
---------------------

Si l'installateur indique que Python 3 est introuvable:

1. Installez Python depuis la page qui s'ouvre.
2. Relancez "install_macos.command".


EN CAS DE PROBLEME
------------------

Si le raccourci "Dars Manager.command" ne s'ouvre pas:

1. Faites clic droit dessus.
2. Cliquez sur "Ouvrir".
3. Confirmez avec "Ouvrir".


SI LE DOUBLE-CLIC NE MARCHE PAS
-------------------------------

1. Ouvrez l'application Terminal.

2. Tapez:

   cd 

   Attention: laissez un espace apres cd.

3. Glissez le dossier dars-manager dans la fenetre Terminal.

4. Appuyez sur Entree.

5. Tapez:

   xattr -dr com.apple.quarantine . && chmod +x install_macos.command drsm_mac.command

6. Appuyez sur Entree.

7. Tapez:

   ./install_macos.command

8. Appuyez sur Entree.
