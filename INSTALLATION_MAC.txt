DARS MANAGER - INSTALLATION SUR MAC
===================================

Ce programme reste local sur votre Mac.
Vos audios, analyses et exports ne sont pas envoyes dans le cloud.


INSTALLATION
------------

1. Dezippez le fichier recu.

2. Ouvrez le dossier dars-manager.

3. Double-cliquez sur:

   Installer Dars Manager.app

4. Si le Mac refuse d'ouvrir l'application:

   - faites clic droit sur "Installer Dars Manager.app";
   - cliquez sur "Ouvrir";
   - confirmez avec "Ouvrir".

5. Une fenetre Terminal peut s'ouvrir.
   Laissez l'installation se terminer.
   Le premier lancement peut prendre plusieurs minutes.

6. Quand l'installation est terminee, un raccourci est cree sur le Bureau:

   Dars Manager.command


UTILISATION
-----------

Pour lancer Dars Manager:

1. Double-cliquez sur le raccourci du Bureau:

   Dars Manager.command

2. L'application s'ouvre dans votre navigateur.

3. Si le navigateur ne s'ouvre pas automatiquement, ouvrez cette adresse:

   http://localhost:8501


OU SONT LES FICHIERS ?
----------------------

Vos fichiers sont stockes ici:

   Documents/DarsManager

Ce dossier contient:

   uploads    : audios charges
   analyses   : analyses sauvegardees
   exports    : audios generes
   hf_cache   : cache du modele Whisper


SI PYTHON EST DEMANDE
---------------------

Si l'installateur indique que Python 3 est introuvable:

1. Installez Python depuis la page qui s'ouvre.
2. Relancez "Installer Dars Manager.app".


EN CAS DE PROBLEME
------------------

Si le raccourci "Dars Manager.command" ne s'ouvre pas:

1. Faites clic droit dessus.
2. Cliquez sur "Ouvrir".
3. Confirmez avec "Ouvrir".
